  • Maca is a website application for keeping a list of books that are currently being read and have been read. And the word "Maca" is taken from the Sundanese language which means "Read".
  • The technology used is HTML, CSS and JavaScript
  • Features :
    1. Add books to bookshelf being read with book details like title, author and year.
    2. See a list of books being read on the shelf.
    3. See the list of finished books on the shelf.
    4. Search for books by title, author and year on the shelf.
    5. Move the book read to finished read and vice versa.
    6. Delete the book if it is no longer needed.
    7. Save book data to local storage in browser.
    8. Switch light or dark theme mode.
    9. Responsive on any device.
    10. The layout created is the result of imagination and self-exploration.
